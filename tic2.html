Réseau (informatique) Network :

Ensemble d’ordinateurs et de terminaux (ou des périphériques)
autonomes (libres) connectés entre eux et qui sont situés dans
un certain domaine géographique, pour échanger des
informations numériques, selon des règles bien définies.

Les caractéristiques de base d ’un réseau sont :

  • La topologie
  • Le débit exprimé en bits/s (ou bps)
  • La distance maximale (ou portée)
  • Le nombre de nœuds maximum

Quelques terminologies en réseau:

  • Station de travail
  • Nœud
  • Paquet
  • Réseaux Hétérogènes
  • Le Débit mesure une quantité de données numériques (bits)

Éléments d'un reseau:

  • Les ordinateurs
  • Les logiciels
  • Les supports du lan
  • Les équipements d'interconnexion du LAN

Classification des réseaux:

  • Par portée l’envergure (la distance) : LAN, MAN et WAN
  • L’ouverture (leur utilisation)
  • Les supports de connexion : filaires et non filaires
  • Le type d’organisation : Peer to Peer et client serveur

Le Modèle OSI:

est un modèle de référence pour les communications
de réseau qui divise les communications en sept
couches distinctes. Chaque couche est responsable
de fonctions spécifiques liées à la communication de données
entre les appareils de réseau.
Le modèle OSI est souvent utilisé comme référence
pour la conception et la mise en œuvre de réseaux informatiques.
Les sept couches du modèle OSI sont les suivantes :

  • La couche physique
  • La couche liaison de données
  • La couche réseau
  • La couche transport
  • La couche session
  • La couche présentation
  • La couche application

L’adressage:

Une adresse IP (Internet Protocol) est le numéro
qui identifie chaque ordinateur dans les réseaux
qu’il appartient. Il y’a deux mode d’adressage IPV4 et IPV6

  • ADRESSE IPV4:

    • Chaque paquet contient : adresse source sur (32 bits)
      et une adresse destinataire sur (32bits)
    • Donc IPV 4 == 4 octets ==32 bits, Par exemple :
      10101100 00010000 00000100 00010100 est exprimé
      en décimale en : 172.16.4.20 -> 4 chiffres en décimale séparé par un point

  • ADRESSE IPV6 :

    • Les adresses sont codées sur 128 bits.
      La représentation textuelle se fait en découpant
      le mot de 128 bits de l'adresse en 8 mots de 16 bits
      séparés par le caractère «:», chacun d'eux
      est en hexadécimal.
      Example :
      FEDC:BA98:7654:3210:EDBC:A987:6543:210F

Principe World Wide Web :

  • Accéder à des documents textuels
